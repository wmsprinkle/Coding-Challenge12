# GreenGrocer Business Analytics Project

This project applies Python-based data analysis techniques to a simulated business scenario for GreenGrocer, an organic grocery store chain. The goal is to extract insights from sales, customer, and operational data to support data-driven decision-making.

## Overview
The analysis covers core areas of business analytics, including descriptive statistics, data visualization, and basic predictive modeling. Python libraries such as Pandas and Matplotlib are used to clean, analyze, and visualize the data.

## Objectives
- Perform descriptive and diagnostic analysis on business data
- Calculate key performance metrics
- Identify trends and relationships between variables
- Create visualizations to support insights
- Apply basic regression for forecasting

## Tools & Technologies
- Python
- Pandas
- Matplotlib

## Key Features
- Data cleaning and preprocessing
- Summary statistics and KPI calculations
- Correlation analysis
- Visualizations (charts and graphs)
- Basic predictive modeling (regression)

## Results
The project identifies patterns in sales and operations, highlights key drivers of performance, and provides simple forecasts to support business planning.

## Purpose
This project demonstrates foundational business analytics skills and the ability to translate data into actionable insights.
